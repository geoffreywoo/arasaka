# Security

This repository is a static website with no application server, database, user accounts, or required environment variables.

## Reporting Issues

If you find a security-sensitive issue, please avoid opening a public issue with exploit details. Contact Geoffrey Woo through the public contact paths linked from https://www.geoffreywoo.com/.

For ordinary bugs, broken links, layout problems, attribution corrections, and copy fixes, GitHub issues or pull requests are welcome.

## Scope

In scope:

- Static-site misconfiguration
- Broken security headers or deployment configuration
- Supply-chain concerns introduced by future tooling
- Accidental exposure of private deployment files

Out of scope:

- Fictional in-world security claims
- Game lore accuracy disputes
- Third-party platform behavior outside this repository
